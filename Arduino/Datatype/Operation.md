#Ardunio 运算符
##
- **赋值运算符**

赋值语句的作用是把某个常量或变量或表达式的值赋值给另一个变量。符号为‘=’。这里并不是等于的意思，只是赋值，等于用‘==’表示。
例如：

count=5;

total1=total2=0;

count的值是5，total1与total2的值都是0。

- **算术运算符**

符号 功能:


' + ' 单目正

' - '  单目负

' * '  乘法

' / '  除法

' % '   取模

' + '   加法

' - '   减法

- **逻辑运算符**

符号 功能：

&&  逻辑与

||    逻辑或

!     逻辑非

- **关系运算符**

' > '   大于

' < '   小于

' >='   大于等于

' <='   小于等于

' =='    等于

' !='    不等于
