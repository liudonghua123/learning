# $()函数

在jQuery中，无论使用那种类型的选择符，都要从一个美元符号和一对括号开始：**$()**。  
这个函数接受CSS选择符作为参数，返回包含页面中对应元素的jQuery对象，所有能在样式表中使用的选择符都可以传给这个函数，随后就可以对匹配的元素集合应用。

> 美元符号$只不过是标识符jQuery的“别名”，为了避免和其他三方库冲突，可以使用**onConflict**进行释放操作。  
 `var jq = $.onConflict();`
